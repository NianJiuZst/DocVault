## ⚠️ 关键风险

### 1. 安全漏洞
`GET /documents/:id` 和 `GET /documents/:id/versions` 没有 owner/share 权限校验，知道文档 ID 的已登录用户理论上就能读取任意文档。

**受影响文件：**
- `packages/backend/src/modules/documents/documents.controller.ts`
- `packages/backend/src/modules/documents/documents.service.ts`

### 2. 部署风险
数据库 schema 和迁移文件不一致，直接部署会失败。

**受影响文件：**
- `packages/backend/prisma/schema.prisma`
- `packages/backend/prisma/migrations/20250925074727_init/migration.sql`

### 3. 协作鉴权过弱
Hocuspocus 只检查 token 非空，不验证 JWT，也不验证文档权限。

**受影响文件：**
- `packages/backend/src/modules/collaboration/collaboration.service.ts`

### 4. 文档宣传超前于实现
README 宣称有表格/KaTeX/图片上传/AI/RAG，但源码都没有实现。

**受影响文件：**
- `README.md`
- `packages/frontend/app/home/cloud-docs/[id]/page.tsx`

### 5. 路由潜在冲突
参数路由 `:id` 写在 `shared/search/tree` 之前，可能被参数路由抢占。

**受影响文件：**
- `packages/backend/src/modules/documents/documents.controller.ts`
